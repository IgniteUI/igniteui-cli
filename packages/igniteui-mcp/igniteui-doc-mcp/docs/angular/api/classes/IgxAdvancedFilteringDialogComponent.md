A component used for presenting advanced filtering UI for a Grid.
It is used internally in the Grid, but could also be hosted in a container outside of it.

Example:
```html
<igx-advanced-filtering-dialog
    [grid]="grid1">
</igx-advanced-filtering-dialog>
```
